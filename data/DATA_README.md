# Dataset Directory

This directory contains the waste classification dataset.

## Structure

```
data/
├── raw/                     # Original, unmodified dataset
│   ├── train/              # Training data
│   │   ├── O/              # Organic waste images
│   │   └── R/              # Recyclable waste images
│   └── test/               # Testing data
│       ├── O/              # Organic waste test images
│       └── R/              # Recyclable waste test images
│
└── processed/              # Preprocessed data (generated by scripts)
    └── (generated files)
```

## Class Labels

- **O** - Organic Waste (biodegradable materials)
- **R** - Recyclable Waste (plastics, metals, paper, etc.)

## Dataset Information

- **Image Format**: JPG/PNG
- **Recommended Size**: 224x224 pixels (will be resized automatically)
- **Color Space**: RGB

## Usage

The dataset is automatically loaded by the training scripts using the paths defined in `config.py`.

### To verify your dataset:

```bash
python config.py
```

This will check that all folders exist and count the images in each class.

## Note for Git

The `raw/` folder is excluded from git tracking due to its size. To share the dataset:

1. **Option 1**: Use Git LFS (Large File Storage)
2. **Option 2**: Use DVC (Data Version Control)
3. **Option 3**: Share via cloud storage (Google Drive, Dropbox, etc.)

## Data Sources

Based on project documents, this dataset appears to be for waste classification following the methodology described in the Computer Vision handbook (Module 6).
